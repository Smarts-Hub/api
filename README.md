# API
This is the Smartshub's main API for all its resources, and web applications such as Pastes.

Powered by TypeScript, Express and MongoDB.

# How to run
1. Install deps
```bash
npm install
```

2. Configure .env
`.env.example --> .env`

3. Run dev
```bash
npm run dev
```
The server is now running on localhost:5000
